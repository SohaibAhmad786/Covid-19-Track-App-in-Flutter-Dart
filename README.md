
# Covid-19 Track App

This app is about the overview of Covid-19 details.

I have used Pie chart, google fonts, animation, shimmer effect, API intergration, loading page and splash screen.

## Packages Used:

- cupertino_icons
- http
- animated_text_kit
- pie_chart
- flutter_spinkit
- shimmer
- google_fonts
